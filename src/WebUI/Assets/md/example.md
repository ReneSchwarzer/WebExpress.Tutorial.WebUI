# Markdown Example

Welcome to this **Markdown Guide**! Below are examples of Markdown syntax to help you structure and style your text.

---

## 1. Text Formatting

- **Bold**: `**text**` → **This is bold text**
- *Italic*: `*text*` → *This is italic text*
- ~~Strikethrough~~: `~~text~~` → ~~This is strikethrough~~

---

## 2. Lists

### Unordered List:
- Item 1
- Item 2
  - Sub-item 2.1
  - Sub-item 2.2

### Ordered List:
1. First item
2. Second item
   1. Nested item 2.1
   2. Nested item 2.2

## 4. Code Blocks

### Inline code:
Use `print(""Hello World!"")` to display text.

### Multiline code:
```python
def greet(name):
    return f""Hello, {name}!""

print(greet(""Markdown""))
```
